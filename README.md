## Récupération du JAR

### Clone le projet en local

```bash
git clone https://github.com/Mathis05000/chatSystem.git
```

### Générer le fichier JAR

```bash
mvn clean package
```

### Exécuter le JAR

```bash
cd front/target
java -jar ./front-1.0-jar-with-dependencies.jar
```
